# Favicon CorpChat

**Ubicación**: Colocar el archivo `favicon.ico` corporativo aquí.

**Especificaciones**:
- Formato: ICO
- Tamaños: 16x16, 32x32, 48x48
- Colores: Usar paleta corporativa

**Generación**:
1. Crear un icono con las iniciales "CC" o el logo corporativo
2. Usar herramienta como https://realfavicongenerator.net/
3. Guardar como `favicon.ico` en este directorio
4. Eliminar este archivo .md

**Temporal**: Por ahora, el sistema usará el favicon por defecto de Open WebUI.
